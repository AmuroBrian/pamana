These are the following format that you could use for this project:
NOTE: JUST COPY PASTE WALANG IIBAHIN DTO KUNDI CONTENT LANG
1. Only Paragraph
Format:
<div class="container">
    <p>CONTENT</p>
</div>

2. Picture with Paragraph
<div class="container">
    <div class="pic__container">
        <img src="../asset/(name ng picture)" class = "picture"><br>
        (References)
    </div>
    <div class="content__container">
        CONTENT
    </div>
</div>

Sa Image example. NOTE: Ang iibahin lang dto is ung (name ng picture) babase sa name ng image file (only jpg and png).
<img src = "../asset/american.jpg" class = "picture">

NOTE: Sa case number 2 Picture with paragraph pede nyo baliktaring ung dalawang div namely: pic__container and content__container.
Example:

<div class="container">
    <div class="content__container">
        CONTENT
    </div>
    <div class="pic__container">
        <img src="../asset/(name ng picture)" class = "picture"><br>
        (References)
    </div>
</div>
